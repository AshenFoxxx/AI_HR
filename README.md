# Telegram HR Matcher Bot

Бот для автоматизации первичного подбора персонала в Telegram.

## Возможности

- **Управление вакансиями**: создание и просмотр вакансий, хранение в `data/vacancies.json`.
- **Анализ резюме**: обработка `.txt` файлов, извлечение навыков и опыта, сохранение в `data/candidates.json`.
- **Сопоставление**: оценка соответствия кандидата вакансиям на основе эмбеддингов модели `cointegrated/rubert-tiny2`, совпадения навыков и опыта.
- **Результаты**: рейтинг топ-3 вакансий и список совпадающих/отсутствующих навыков.
- **Числовая диагностика**: режим оценки пригодности кандидата по набору числовых признаков (опыт, количество навыков, возраст, образование, match-score) с помощью ML-модели.

## Запуск

1. Установите зависимости:

```bash
pip install -r requirements.txt
```

2. Создайте файл `.env` в корне с содержимым:

```bash
TELEGRAM_BOT_TOKEN=ваш_токен_бота
```

3. Запустите бота:

```bash
python -m src.main
```

## Обучение ML-модели (диагностика кандидата)

Для числовой диагностики (режим "experience=5, skills=8, age=30, education=1, match_score=70") используется отдельная ML-модель.

1. Обучите модель:

```bash
python -m src.train_model
```

Скрипт:

- генерирует синтетический датасет кандидатов с признаками:
  - `experience_years` — опыт работы (0–15 лет),
  - `skills_count` — количество навыков (1–20),
  - `age` — возраст (20–60 лет),
  - `education` — 0 (нет высшего), 1 (есть высшее),
  - `match_score` — процент совпадения с требованиями (0–100);
- целевая переменная: `fit_level` (0 — не подходит, 1 — сомнительно, 2 — подходит);
- обучает и сравнивает 4 модели:
  - LogisticRegression,
  - RandomForestClassifier,
  - XGBoostClassifier,
  - CatBoostClassifier;
- выводит таблицу с метриками Accuracy и F1-macro;
- выбирает лучшую модель по F1-macro и сохраняет её в `models/candidate_model.pkl`;
- сохраняет скейлер признаков в `models/scaler.pkl`.

Пример таблицы (значения могут отличаться на вашем запуске):

```text
Model              |   Accuracy |  F1-macro
-------------------------------------------
LogisticRegression |     0.78   |     0.77
RandomForest       |     0.85   |     0.84
XGBoost            |     0.88   |     0.87
CatBoost           |     0.87   |     0.86
```

В примере выше наилучший F1-macro показывает модель **XGBoost**, поэтому именно она сохраняется как основная модель для числовой диагностики кандидатов.

## Важно

Бот носит **рекомендательный** характер и не заменяет работу HR-специалиста. Всегда проводите ручную проверку кандидатов.
